# Code of Conduct

Waypoint follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The two local additions:

1. **Elmering is the culture.** Amateur radio has a long tradition of experienced operators mentoring newcomers. Questions that would be "RTFM" elsewhere get a pointer and a welcome here.
2. **Other projects are off-limits as targets.** Criticism of Pi-Star, WPSD, or any other project's code or maintainers doesn't belong in this space. We compare architectures, not people.

Reports: conduct@ (see MAINTAINERS.md) or via private message to any maintainer. Reports are handled confidentially.
