Waypoint

An open, community-governed host system and UI for MMDVM digital voice hotspots — DMR, YSF, D-Star, P25, NXDN, POCSAG — built on the public g4klx GPL stack and its new MQTT data plane.

GitHub Requirements register Governance

Phase 0 — foundation

Nothing is usable yet. But everything is public from the first commit: the architecture, the governance commitments, and a requirements register where every item cites the real community complaint that motivated it.

The promises

Lossless configuration

One schema-versioned store; gateway INIs are generated artifacts. Applying a change never destroys another setting.

Honest status

Structured MQTT events, not log scraping. If the dashboard says linked, it's linked.

Secure by default

No default credentials, HTTPS out of the box, a real private security-reporting channel.

Safe updates

Tagged, changelogged, signed. Atomic or they don't happen. Your customizations survive in a documented override layer.

Phone-first UI

Responsive, dark-mode default, screen-reader accessible — merge gates, not aspirations.

Governance that outlives people

14-day review SLA, RFC process, a written no-telemetry policy, and a stated multi-maintainer goal.

Contribute

Every PR gets a human response within 14 days — even if it's "no, and here's why." AI-assisted triage gives you a technical first read within minutes; maintainers make the decisions. Start with the curated first issues or read CONTRIBUTING.md.
